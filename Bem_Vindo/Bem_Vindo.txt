Bem-vindo!

É um prazer tê-lo como aluno do nosso curso online SQL 2016 - Criando Sistemas de Banco de Dados. Estecurso é
ideal para você que deseja aprender a criar e manipular um sistema de banco de dados em SQL Server.

Durante o curso, você aprenderá a criar um banco de dados e utilizar diversos recursos do SQL Server e sua estrutura
de consulta e manipulação de dados. Bom aprendizado!
